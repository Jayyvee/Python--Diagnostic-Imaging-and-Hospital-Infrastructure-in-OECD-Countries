# Diagnostic Imaging and Healthcare Infrastructure in OECD Countries

# Project Overview
This project analyzes how diagnostic imaging capacity, hospital infrastructure, and hospital stay duration have evolved across OECD countries. Using data from OECD Health Statistics, the study explores relationships between technology investment and healthcare efficiency.

# Research Question
How does investment in diagnostic imaging technology relate to hospital infrastructure and healthcare efficiency across OECD countries?

# Hypothesis
Increased diagnostic imaging capacity is associated with expanded healthcare infrastructure while contributing to shorter hospital stays and improved system efficiency

# Dataset
Source: OECD Health Statistics
Variables used:
* CT scanners per million population
* Hospital beds per 1,000 population
* Average hospital stay length
* Country and year indicators

# Tools Used
* Python (pandas, seaborn, matplotlib)
* Tableau
* Jupyter Notebook
* OECD Health Statistics dataset

# Analysis Workflow
* Data cleaning and merging
* Time-series analysis of healthcare infrastructure
* Correlation analysis between CT Scanners and hospital beds
* Outlier Analysis
* Cluster modeling of healthcare system types
* Dashboard visualization in Tableau

# Key Findings
* CT Scanner availability increased significantly across OECD countries from 1990-2018
* Countired with greater diagnostic imaging
